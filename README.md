# Heart Disease Prediction System

. Project Overview
This project predicts the risk of heart disease using Machine Learning techniques. The model is trained on patient health data and provides predictions based on medical attributes.

. Features
- Heart disease risk prediction
- Interactive Streamlit web application
- Machine Learning based prediction model
- User-friendly interface

. Technologies Used
- Python
- Pandas
- NumPy
- Scikit-Learn
- Streamlit
- Joblib

. Project Files
- app.py
- heart.csv
- logisticRegression_heart.pkl
- scaler.pkl
- columns.pkl

.  How to Run
1. Install required libraries:
   pip install streamlit pandas numpy scikit-learn joblib

2. Run the application:
   streamlit run app.py
